dataset.base module

class ResizableDataset(images_path, annotations_path, cfg)

Bases: Dataset, ABC

Abstract base class for resizable datasets.

This class defines a framework for datasets that support resizing images and their corresponding annotations. It must be subclassed to implement specific dataset formats.

abstract reshape(img, anns, shape, resize_image_method)

Reshape an image and its associated annotations by a specified factor.

This function adjusts the size of the input image and modifies the annotations accordingly to maintain their relevance after resizing. The image can be resized using different methods specified by the user.

Parameters:

  • img (np.ndarray) – The image to be scaled, represented as a NumPy array.

  • anns (list) – A list of annotations associated with the image, which will also be scaled.

  • shape (tuple) – The new shape of the image and annotations.

  • resize_image_method (str) – The method used for resizing the image.

Returns:

A tuple containing the scaled image (np.ndarray)

and the modified annotations (list).

tuple

abstract scale(img, anns, scale_factor, resize_image_method)

Scales an image and its associated annotations by a specified factor.

This function adjusts the size of the input image and modifies the annotations accordingly to maintain their relevance after scaling. The image can be resized using different methods specified by the user.

Parameters:

  • img (np.ndarray) – The image to be scaled, represented as a NumPy array.

  • anns (list) – A list of annotations associated with the image, which will also be scaled.

  • scale_factor (float) – The factor by which to scale the image and annotations.

  • resize_image_method (str) – The method used for resizing the image.

Returns:

and the modified annotations (list).

Parameters:
